# AAL-Band-GUI-Setup

Desktop application and User Guide for AAL-Band.

## Download
Download the User Guide from main branch.

Download the installer from the GitHub Release page:

1. Open the `v0.1.0` release:
	https://github.com/BioxGroup/AALBand-GUI-Setup/releases/tag/v0.1.0
2. In **Assets**, download:
	- `BioX AAL-Band GUI Setup.exe`

## Install (Windows 10/11)

1. Close any previous running instance of the app.
2. Run the downloaded installer (`.exe`).
3. Follow the setup wizard.

## First run note (not code-signed yet)

Because the installer is not code-signed yet, Windows may show a warning such as **“Windows protected your PC”**.

To complete the installation:

1. Select **More info**.
2. Select **Run anyway**.

Some managed computers enforce policies that block running unsigned applications. If you do not see the “Run anyway” option, contact your IT administrator.

## Uninstall

Use **Settings → Apps → Installed apps** (Windows 11) or **Control Panel → Programs and Features** (Windows 10) to remove the application or you can find the program uninstall file in this directory : "C:\Program Files (x86)\BioX AAL-Band GUI"

## Support

If you encounter installation issues, please include:

- Your Windows version
- The installer version (release tag)
- A screenshot or exact text of any Windows/IT security prompt
